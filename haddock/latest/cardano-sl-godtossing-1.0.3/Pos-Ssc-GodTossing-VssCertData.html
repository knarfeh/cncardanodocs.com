data VssCertData Source #

Wrapper around VssCertificate with TTL. Every VssCertificate has own TTL. Wrapper supports simple HashMap operations. Wrapper holds VssCertificatesMap and Set of certificates sorted by expiry epoch.

Fields

  • lastKnownEoS :: !EpochOrSlot

    Last known slot, every element of expirySlotSet > lastKnownEoS

  • certs :: !VssCertificatesMap

    Not expired certificates

  • whenInsMap :: !(HashMap StakeholderId EpochOrSlot)

    Slot when certs was inserted. It is needed for deletion from insSlotSet (by StakeholderId).

  • whenInsSet :: !(Set (EpochOrSlot, StakeholderId))

    Set of pairs (insertion slot, address hash) Every element of insSlotSet <= lastKnownEoS

  • whenExpire :: !(Set (EpochOrSlot, StakeholderId))

    Set of pairs (expiry slot, address hash). Expiry slot is first slot when certificate expires. Pairs are sorted by expiry slot (in increasing order, so the oldest certificate is first element).

  • expiredCerts :: !(Set (EpochOrSlot, (StakeholderId, EpochOrSlot, VssCertificate)))

    Set of expired certs for current lastKnownEoS. We store only certificates which expried no earlier than in previous epoch. Set (full expired slot, (id, insertion slot, cert))
