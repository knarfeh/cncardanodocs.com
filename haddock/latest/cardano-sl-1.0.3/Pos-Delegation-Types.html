newtype DlgPayload Source #

DlgPayload is put into MainBlock and consists of a list of heavyweight proxy signing keys. There must be no duplicates (comparing by issuer) in this list. The order of PSKs doesn't matter, as it's checked for cycles after bulk application. So it's technically a set.
