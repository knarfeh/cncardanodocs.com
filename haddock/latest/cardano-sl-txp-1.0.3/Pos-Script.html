data PlutusError Source #

The type for errors that can appear when validating a script-protected transaction.

Constructors

PlutusUnknownVersion ScriptVersion

A script has a version that we don't know how to execute.

PlutusDecodingFailure Text

A script couldn't be deserialized.

PlutusExecutionFailure Text

The script evaluator refused to execute the program (e.g. it doesn't typecheck, or the evaluator has run out of petrol).

PlutusException Text

The script evaluator threw an exception (e.g. with error).

PlutusReturnedFalse

Everything typechecks and executes just fine but the result of evaluation is failure and so the transaction is invalid.
