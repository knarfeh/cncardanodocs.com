shaStressRedeemer :: Int -> Script Source #

Does N rounds of SHA3-256. Should be used with idValidator.

Actually it does (N div 10) * 10 rounds. Unrolling the loop is done so that more petrol would be spent on hashing and less – on substraction and function calls.
