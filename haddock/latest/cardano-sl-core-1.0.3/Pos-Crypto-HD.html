data HDAddressPayload Source #

HDAddressPayload consists of

  • serialiazed and encrypted with symmetric scheme path from the root key to given descendant key with passphrase (using ChaChaPoly1305 algorithm)
  • cryptographic tag

For more information see packHDAddressAttr and encryptChaChaPoly.
