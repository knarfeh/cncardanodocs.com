fromAvvmPk :: MonadFail m => Text -> m RedeemPublicKey Source #

Read the text into a redeeming public key. The key should be in AVVM format which is base64(url). This function must be inverse of redeemPkB64UrlF formatter.

There's also a copy of this function in cardano-addr-convert.
