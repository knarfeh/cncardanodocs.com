newtype UnreasonableEoS Source #

Wrapper over EpochOrSlot. Its Arbitrary instance is made to guarantee its EpochIndex is in the interval (maxReasonableEpoch, maxBound :: Word64 ]. This is to ensure the property 'toEnum . fromEnum = id' holds for all EpochOrSlot, not just the ones whose EpochIndex uses the "reasonable" Arbitrary instance.
