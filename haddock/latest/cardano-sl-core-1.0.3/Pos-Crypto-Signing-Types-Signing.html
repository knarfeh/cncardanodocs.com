data ProxySignature w a Source #

Delegate signature made with certificate-based permission. w stays for message type used in proxy (ω in the implementation notes), a for type of message signed.

We add whole psk as a field because otherwise we can't verify sig in heavyweight psk transitive delegation: i → x → d, we have psk from x to d, slot leader is i.
