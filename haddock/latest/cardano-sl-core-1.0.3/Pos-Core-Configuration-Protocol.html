blkSecurityParam :: HasProtocolConstants => BlockCount Source #

Security parameter which is maximum number of blocks which can be rolled back. This value is embedded into library and can be used only for initialization. The actual value should be fetched from runtime context (it can differ from this one).

slotSecurityParam :: HasProtocolConstants => SlotCount Source #

Security parameter expressed in number of slots. It uses chain quality property. It's basically blkSecurityParam / chainQualityThreshold.
