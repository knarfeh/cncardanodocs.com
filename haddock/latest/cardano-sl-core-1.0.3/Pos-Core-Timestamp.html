newtype Timestamp Source #

Timestamp is a number which represents some point in time. It is used in MonadSlots and its meaning is up to implementation of this type class. The only necessary knowledge is that difference between timestamps is microsecond. Hence underlying type is Microsecond. Amount of microseconds since Jan 1, 1970 UTC.
