Description

Pure database implementation using Map. More efficient option would be something akin to muteble prefix tree. http://hackage.haskell.org/package/bytestring-trie might do better though it's immutable.

Pure database datatype. Includes 4 subdatabases mocking rocks dbs and a map for storing blocks data (we store them in files directly in real implementation).
